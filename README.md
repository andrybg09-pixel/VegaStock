# VegaStock

Aplicación móvil de gestión de inventario en tiempo real para **Mini Super Vega**.

VegaStock es una aplicación Android desarrollada para **Mini Super Vega**, un negocio familiar
ubicado en Barrio Tulita Sandino, Santa Cruz, Guanacaste. La app permite controlar el inventario
en tiempo real, recibir alertas de stock bajo y generar reportes de movimientos, reemplazando el
control manual que el negocio usa actualmente.

---

## Problema que resolvemos

El negocio lleva el inventario de forma manual, lo que genera:

- Falta de visibilidad del stock en tiempo real.
- Sin reportes de ganancias/pérdidas ni de movimientos.
- Ingreso lento de productos cuando llegan los proveedores.

## Solución (MVP)

- Inicio de sesión y manejo de sesiones.
- Listado dinámico de productos por categoría con búsqueda.
- Registro de productos con validaciones (SKU, precios, stock mínimo, unidad de medida).
- Registro de entradas y salidas de inventario con historial de movimientos.
- Alertas de stock crítico y por agotarse.
- Reportes: total de productos, valor de inventario, movimientos del mes y productos con menor stock.
